# webpack2基本配置



## 文件目录详情

### build

1. webpack的基本配置文件

### config (配置基本变量)



### dist

1. 打包后生成的文件

### src (编写源代码的文件)

1. component (放置vue的组件文件)
2. vuex (是vue的一个状态管理文件)
3. images (放置图片)
4. js (封装好的js文件)
5. css (公共css)
6. router (vue的路由-核心功能)
7.  app.vue (入口根组件)

### static

1. 放置一些静态资源

index.html (编译html)

package.json (核心文件 安装依赖 执行脚本)

main.js (入口js)

webpack.config.js (配置loader和plugin的入口文件)
